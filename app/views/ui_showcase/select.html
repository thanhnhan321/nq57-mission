Reload options with `max` (custom event)

The “property type” select refetches options when the dependency select fires select:reload:property-type. The event payload sets max, limiting how many options the API returns.

{{ select( name='property_type_reload_demo', options_url=url('ui_showcase_select_options'), label='Kiểu tài sản (max-limited)', placeholder='Chọn kiểu tài sản', reload_events=['select:reload:property-type'], ) }} {{ select( name='dependency', options_url=url('ui_showcase_dependency_max_options'), label='Max options', placeholder='Chọn max', value=none, extra_attributes={ '@change': "$dispatch('select:reload:property-type', { max: Array.from($event.target.selectedOptions).map(option => option.value) });", } ) }}
