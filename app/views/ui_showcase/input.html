Date / Time Picker

Three variants returning ISO8601 values: date (`YYYY-MM-DD`), time (`HH:mm:00`), and datetime (`YYYY-MM-DDTHH:mm:00`). Optional bounds: pass `min` and `max` in `extra_attributes` as the names of string properties on an ancestor Alpine `x-data` (ISO date values). The picker exposes getters min / max (Date | null; null = unbounded). Sibling pickers resync when a hidden input dispatches a bubbling change to the outermost x-data root.

Date picker

{{ datetimepicker( name='demo_date_picker', variant='date', label='Ngày', hint='ISO: YYYY-MM-DD', value='2026-03-23' ) }}

Time picker

{{ datetimepicker( name='demo_time_picker', variant='time', label='Giờ', hint='ISO: HH:mm:00', value='10:01:00' ) }}

Date time picker

{{ datetimepicker( name='demo_datetime_picker', variant='datetime', label='Ngày giờ', hint='ISO: YYYY-MM-DDTHH:mm:00', value='2026-03-23T10:01:00' ) }}

Date picker (month + year only)

Fix `day=1`, let users choose only month and year.

{{ datetimepicker( name='demo_date_picker_month_year_only', variant='date', label='Tháng áp dụng', hint='Selectable: year + month. ISO output is still full date.', value='2026-03-01', extra_attributes={ 'day': 1, 'format': 'Tháng MM/YYYY' } ) }}

Date picker (year only)

Fix `month=1` and `day=1`, let users choose only year.

{{ datetimepicker( name='demo_date_picker_year_only', variant='date', label='Năm', hint='Selectable: year only. ISO output stays YYYY-01-01.', value='2026-01-01', extra_attributes={ 'month': 1, 'day': 1 } ) }}

Date bounds via parent keys (min / max)

Parent holds ISO strings; extra_attributes passes property names feb_min_iso / feb_max_iso.

{{ datetimepicker( name='demo_date_picker_bounded_static', variant='date', label='Trong tháng 2/2026', hint='đọc min/max từ parent', value='2026-02-15', extra_attributes={ 'min': 'feb_min_iso', 'max': 'feb_max_iso', } ) }}

Date bounds from parent x-data

End date cannot be before start; start cannot be after end. Parent keys demo_range_start / demo_range_end update on each picker’s hidden input change.

{{ datetimepicker( name='demo_date_range_start', variant='date', label='Từ ngày', hint='max = demo_range_end', value='2026-01-10', extra_attributes={ 'max': 'demo_range_end', } ) }} {{ datetimepicker( name='demo_date_range_end', variant='date', label='Đến ngày', hint='min = demo_range_start', value='2026-06-20', extra_attributes={ 'min': 'demo_range_start', } ) }}

Time bounds via parent keys

Only times between 09:30 and 17:00 are selectable.

{{ datetimepicker( name='demo_time_picker_bounded', variant='time', label='Giờ làm việc', hint='min/max đọc từ parent', value='10:01:00', extra_attributes={ 'min': 'demo_time_min', 'max': 'demo_time_max', } ) }}

Datetime range (same day)

End cannot be before start; bounds include time. Try changing start/end to see the time list disable.
